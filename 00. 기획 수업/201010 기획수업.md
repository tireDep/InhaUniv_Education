## UML
- Unsigned Modeling Language
-- 통합된 모델링 언어(본, 형틀)
-- Alphabet + Grammar

- 클래스 != 객체
	- 클래스 
	-- 객체의 설계도
	-> 명사로 작성

	- 객체 
	-- 클래스의 인스턴스
	-- 동사로 작성
	-- 멤버 변수
	-> 명사로 작성
	-- 멤버 함수 
	-> Function(독립된 기능), Method(클래스에 종속된 기능)
	-> 동사로 작성

- 클래스 다이어그램
-- private 부분 굳이 작성은 필요하지 않음, Memo로 따로 빼기
-- Get, Set 굳이 명시 하지 않아도 됨
-- 상속 시 함수가 동일하면(부모함수 사용) 굳이 명시하지 않아도 됨
